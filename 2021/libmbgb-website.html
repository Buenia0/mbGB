Welcome to the new libmbGB!

April 2, 2021 – written by BueniaDev

I would like to formally welcome you to libmbGB's brand-new website! This new site has been in the works off and on for over a year now. A lot of hard work, and a few false starts, but here we are! Thanks to Milwaukee Area Technical College for motivating me enough to finally get this site online!

Some of the features of libmbGB include:

  • Solid Game Boy (and Game Boy Color) emulation.
  • Real-time clock support for games that use the RTC (such as the Gen 2 Pokemon games).
  • Custom bootroms, removing the need to use seperate bootroms. Seperate bootroms can still be used, if desired, for additional accuracy.
  • A powerful SDL2 reference frontend with both software and hardware-accelerated OpenGL backends, controller and controller hot-plugging support, and more!

Even more featues are currently in the works, although they aren't quite ready yet:

  • Rewrite of link cable emulation, supporting both local and network multiplayer, as well as Mobile Adapter GB, Game Boy Printer, and Barcode Boy support.
  • Cheat code support.
  • A libretro core, compatible with RetroArch.
  • Re-recording support and custom scripting for tool-assisted speedruns.
  • A fully featured debug suite.

Furthermore, feature requests are welcome at this point, if you have a brilliant idea that's not on the above list. The source code for libmbGB is available on GitHub under the GNU GPL v3.0 right now.

libmbGB. The best is yet to come.
